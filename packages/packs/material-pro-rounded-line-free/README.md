# @icon-pkg/streamline-material-pro-rounded-line-free

Redistributed Streamline icon pack for Streamline Material Rounded Line Free.

- Family: Streamline Material
- Style: Rounded Line
- Icons: 50
- Source: https://www.streamlinehq.com/icons/material-pro-rounded-line-free
- Browse icons: https://sebastian-software.github.io/effective-icon/packs/material-pro-rounded-line-free/
- License: CC BY 4.0
- Redistributor: Sebastian Software GmbH, Mainz, Germany
- OSS Home: https://oss.sebastian-software.com

## Install

```bash
npm install @icon-pkg/streamline-material-pro-rounded-line-free
```

## Contents

- `manifest.json` for pack metadata and icon lookup
- flat `icons/*.svg` files for downstream tooling and asset access
- `index.html` for a static icon gallery grouped by category

This package redistributes the publicly available Streamline free icon set and is intended to be consumed by build tools such as `@effective/icon`.
